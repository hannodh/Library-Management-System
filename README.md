# 📚 Library Management System

A simple Library Management System developed in Java using Object-Oriented Programming (OOP).

## Features

- Add a new book
- Display all books
- Search for a book by ID
- Borrow a book
- Return a book

## Technologies Used

- Java
- NetBeans IDE
- Object-Oriented Programming (OOP)

## Project Structure

```
LibraryManagementSystem/
│
├── Book.java
├── Library.java
└── Main.java
```

## How to Run

1. Open the project in NetBeans.
2. Run `Main.java`.
3. Follow the menu displayed in the console.
